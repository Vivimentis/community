# Vivimentis Community

Welcome to the official Vivimentis Community.

This repository serves as the public hub for development updates, roadmaps, announcements, and community discussions across all Vivimentis products.

## About Vivimentis

**Vivimentis** is an independent software studio focused on creating thoughtful digital experiences that are simple, accessible, and enjoyable.

Our mission is reflected in our slogan:

**Play Smart. Live Bright.**

We build products that help people learn, think, create, and enjoy everyday moments through technology.

---

## Our Products

### MemoVivid

A memory matching puzzle game designed to challenge concentration, observation, and recall skills.

### Bulls & Cows Number Puzzle

A classic logic deduction game that encourages analytical thinking and problem-solving.

### QR Code Generator - QR Creator

A practical QR code creation tool designed for everyday use.

More products are currently in development.
